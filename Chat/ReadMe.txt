1.To connect the server type "/server" , after the connection it must say "connected!!".

2.To change your nickname type "/nick fuu" , "fuu" is the name you choose.

3.To view the list of all the current users type "/who".

4.To upload a file type "/upload filename.format +" , "filename" is the name of your file and "format" is the format of the file you want to upload.

5.To view the list of the downloadable files type "/ls".

6.To download a file type "/download filename.format" , same as upload but without "+" , the file has to be from the downloadable files.

7.And to send amsg just type it ;) .
